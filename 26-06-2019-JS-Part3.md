# JavaScript

## Print from -1 to 10

```js
for (var i = -1; i < 11; i++) {
    if (i % 2 == 0) {
        console.log(i + " is even")
    }
    else {
        console.log(i + " is odd")
    }
}
```

## Print from 1 to 1000 with the least lines of code

```js
for (var i=0;i<=1000;i++) {console.log(i)}
```
